Group 3's implementation of a program that tracks moving shipment trucks.

## Technology Stock
- ElectronJS (Infrastructure)
- Bootstrap (Design)
- Axios (HTTP Requests)
- AWS (DB)

## How to run 
1. Download/clone repo (zip)
2. Extract zip file
3. `cd` into desktop and folder
4. run `npm install` then run `npm run`
